# Frontend-Einstellungen (fuer normale Nutzer)

Hier steuerst Du, wie Deine Kleinanzeigen-Seiten aussehen.

## Presets (am einfachsten)

Du kannst mit einem Klick einen Stil vorwaehlen:

- B2C: schnell, kompakt, viele Karten
- Premium: edler, mehr Vertrauen, grosszuegiger
- Community: reduziert, ruhiger

Wenn Du unsicher bist: Starte mit B2C und passe danach einzelne Schalter an.

## Bereich "Archiv" (Anzeigen-Liste)

Wichtige Schalter:

- Letzten Filter automatisch laden
- Anzahl Spalten
- Schnellansicht
- Merken-Button
- Kontakt-Button in der Karte
- "Reserviert"-Badge

Empfehlung:
- 3 Spalten fuer die meisten Seiten.
- Schnellansicht aktiv, wenn Du schnelle Vergleiche willst.

## Bereich "Single" (Detailseite)

Wichtige Schalter:

- Galerie anzeigen
- Verkaeuferbox anzeigen
- Sticky Aktionen auf Mobilgeraeten
- Reserviert-Badge
- Akzentfarbe

Wenn Deine Seite serioes wirken soll:
- Galerie + Verkaeuferbox aktiv
- ruhige Akzentfarbe nutzen

## Medienlayout pro Stil

Du kannst je Stil festlegen:

- Hero oben: Einzelbild / Slider / Mosaik
- Zusatzbilder: ueber oder unter der Beschreibung
- Zusatzbilder als Grid oder Slider

Praxis-Tipp:
- Viele Bilder: Slider
- Wenige Bilder: Grid

## Bereich "User"

- Intro-Text im Nutzerbereich
- Tab "Gemerkte Anzeigen"
- Reservieren im Nutzerbereich erlauben
- Tarif-Status-Box ein/aus + Design

Wenn Du Credits nutzt: Tarif-Status-Box einschalten.

## Schnelles Set-up (empfohlen)

1. Preset waehlen
2. 3 Spalten
3. Galerie + Verkaeuferbox aktiv
4. Reserviert-Badge aktiv
5. Speichern
6. Eine echte Test-Anzeige im Frontend pruefen
