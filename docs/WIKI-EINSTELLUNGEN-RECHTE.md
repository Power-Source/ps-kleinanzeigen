# Rechte (wer darf was?)

Hier bestimmst Du, welche Benutzerrolle was im Plugin darf.

## So gehst Du vor

1. Rolle waehlen (z. B. Subscriber, Editor)
2. Gewuenschte Haken setzen
3. Speichern

## Wofuer ist das wichtig?

Damit nicht aus Versehen zu viele Rechte vergeben werden.

Beispiel:
- Normale Nutzer: Anzeigen erstellen/bearbeiten
- Redakteure/Admins: freigeben, verwalten, loeschen

## Empfehlung

- Immer mit einer kleinen Testrolle pruefen.
- Erst danach auf echte Nutzerrollen anwenden.
