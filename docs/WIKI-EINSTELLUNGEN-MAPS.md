# PS Maps Integration (einfach)

Nutze diese Seite nur, wenn Du Karten fuer Regionen anzeigen willst.

## Was Du hier einstellen kannst

- Regionen-Karte ein/aus
- Wie viele Anzeigen im Marker-Popup stehen
- Karte auf der Detailseite ein/aus
- Titel ueber der Karte
- Auto-Geocoding (fehlende Koordinaten automatisch ermitteln)
- Standard-Zoom

## Typische Einstellungen

Fuer den normalen Start:

1. Regionen-Karte aktivieren
2. Vorschau pro Marker: 3 bis 5
3. Single-Karte aktivieren
4. Zoom: 6 bis 8
5. Speichern und testen

## Wenn keine Karte erscheint

1. Ist PS Maps wirklich aktiv?
2. Haben Deine Regionen Begriffe?
3. Hat mindestens eine Anzeige eine Region?
4. Ist die Regionen-Karte im Tab aktiviert?
